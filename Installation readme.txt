Installation
a) Go to Run
b) Type: %AppData%/VAT-Spy Press enter
c) Copy the VAT-Spy Data files (FIRBoundaries & VATSpy) & Paste both files in the VAT-Spy folder
d) Overwrite? Yes


Note - If any error like this VATSPY -- ERROR LOADING CONFIG comes you can fix it by following these steps: 
a) Go to Run
b) Type: %AppData%/VAT-Spy Press enter
c) Then delete the file called VATSpyConfig.xml (you might not see the .xml extension) 
d) Restart VAT-Spy
